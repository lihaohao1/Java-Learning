顺序容器ArrayList

1.创建

ArrayList notes = new ArrayList();

类似C中的链表，以存储String类型的数据为例

2.简单功能

  • 添加数据 .add(s)
  • 删除数据 .remove(index)
  • 插入数据 .add(location,s)
  • 取某位值 .get(index)
  • 元素个数 .size()
  • 判断是否包含String .contain("String")
  • 判断List是否为空 .isEmpty()
  • 清空List .clear()

3.toArray(a)用法

  1. String[] a = new String[arrayList.size()];
  2. arrayList.toArray(a);

4.遍历

  • for each循环
  1. for(String s:arryList)
  2. {   
  3.     System.out.println(s);
  4. }
  • 索引值遍历(效率最高)
  1. for(int a = 0; a < arrayList.size(); a++)
  2. {
  3.     System.out.print(arrayList.get(a) + " ");
  4. }
  • 迭代器遍历
  1. Iterator<String> a = arrayList.iterator();
  2. while(a.hasNext())
  3. {
  4.     System.out.print(a.next() + " ");
  5. }
